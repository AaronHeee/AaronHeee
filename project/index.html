SOCIAL NETWORK

GithubEye: A Social-network-based Search Engine for Github
A content search service named GithubEye to help people find Github repos. Type your Github user name and your natural language description of what you are looking for. Our service returns you the sorted repo list based on your social relationship in github. This's a competition project in HACKxFDU, and contributor list is here
Recommender System for Douban Movie (In Chinese)
A system in order to recommend movies for users based on Factorization Machines, exploiting the social network information, comments information and movies rating history in Douban Movie (a famous website in China like IMDb). This is a social network course project cooperated with Shiran Ran and Donghao Li. The report in Chinese is here.
Stereotype Anlysis in NetEase News Comments (In Chinese)
The phenomenon about Regional Stereotype in China is universal (here is an introduction), especially from the comments in NetEase News . We analyze this phenomenon and the province-wise relationships, by mining the stances and sentiments from NetEase News comments. This is a social media course project cooperated with Shiran Ran and Zhishuo Zhou. The report in Chinese is here.

COMPUTER VISION

Large-scale Satellite Imagery Feature Detection
Explore some segmentation models for satellite imagery feature detection such as u-net. Fine tune basic models and explore the best size for satellite imagery. I maintain this project for Citory Ltd. as an intern, but for some commerce reasons, details cannot be provided. This project is supervised by Dr. Fan Zhang.

3D Free Form Deformation (In Chinese)
Apply the algorithm about 3D Free Form Deformation (FFD) with the toolkit VTK. Plus, utilizing the 3D Face Resruction model PRNet, we realize the workflow of 2D-Face -> 3D-Face -> 3D-FFD. It's a course project about visualization, working with Shiran Ran and Yikai Wang. The report in Chinese is here.

OTHERS

Intelligent Agent: Pac Man Search
Adapted from the Berkeley Pac-Man Assignments originally created by John DeNero and Dan Klein. This is an AI course project to implement an agent to undertake specifical tasks, as well as to find the path to the end in mazes. What we do are: 1) implement path-search algorithms for DFS, BFS, UCS as well as A*, and 2) design effective heuristics. The report is here.
ScholarDB: A task/questionnaire managing and analyzing system
A website designed for scholars to deliver questionnaires and dataset annotation tasks, the idea is sparked by Amazon Mechanical Turk . This is a database course project cooperated with Xisen Jin.
Y86 CPU Assembler and Simulator
The course project is about Y86 CPU based on pipeline CPU architecture in CS:APP, also I designed the concurrent version of Y86 CPU simulator named Y86 v2.0 with Linyang He.
